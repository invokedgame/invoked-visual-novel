**Invoked** is a psychological **visual novel** inspired by classic **creepypasta** stories. It combines romance, horror, emotional storytelling, and branching narrative choices into an immersive character-driven experience.

## About the Game

Invoked is a story-driven visual novel where every decision influences relationships, dialogue, and story progression. Inspired by internet horror and creepypasta culture, the game blends emotional storytelling with psychological tension and multiple narrative routes.

Current Version

- Chapter 1 playable
- Additional routes including **CLOCKWORK** and **KATE** are currently in development.
- **Hoodie** and **Brian** routes are planned for future updates.
